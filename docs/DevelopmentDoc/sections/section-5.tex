\section{植物系统}

原版植物生长仅需要土地，光照等简单条件，且仅含一次性收获(小麦)与藤曼生长(南瓜)两种方式，较为简单。模组的植物通过脚本控制，添加了植物种植条件:生态，攀爬。增添了可多次收获植物，树木...以丰富玩法。

\subsection{粮食}

粮食的特点: 多产，提供大量饱食度，无 buff。
