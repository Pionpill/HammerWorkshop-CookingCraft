\noindent\textbf{前言：}

此文档为开发人员(主要为程序员)设计说明书，旨在给予模组设计统一的方向。同时，其它开发人员也可阅读此文档的部分内容，了解设计方向。

本文所有数值设计，模型设计以及逻辑关系均以基岩版为准，暂不考虑开发 Java 版。

\noindent\textbf{注：}
